Контроллеры для пользователей уровня ADMIN:

AdminCategoryController /admin

Получить категорию GET /category

Параметры

  • title
  • price
  • category_id
Обновить категорию PUT /category/update

Параметры

  • category
Удалить категорию DELETE /category/{id}

Параметры

  • id

  • id

  • category_id

Параметры

  • id
